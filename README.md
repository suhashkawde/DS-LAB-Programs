## How to Run

To run the data science programs, please follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/suhashkawde/DS-LAB-Programs.git
   ```

2. Navigate to the desired experiment directory:
   ```bash
   cd DS-LAB-Programs/experiments/experiment1
   ```

3. Run the Python script:
   ```bash
   python script1.py
   ```

4. Make sure to have the required libraries installed. You can do this by running:
   ```bash
   pip install -r requirements.txt
   ```
