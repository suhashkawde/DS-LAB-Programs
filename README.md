## 🚀 How to Run

### Prerequisites
- GCC compiler installed
- Basic knowledge of C programming
- Text editor or IDE (VS Code, Code::Blocks, etc.)

### Steps to Compile and Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/suhashkawde/DS-LAB-Programs.git
   cd DS-LAB-Programs
   ```

2. **Navigate to a practical folder:**
   ```bash
   cd "DS Programs/Practical 1"
   ```

3. **Compile the C program:**
   ```bash
   gcc -o practical1 practical1.c
   ```

4. **Run the executable:**
   ```bash
   ./practical1      # On Linux/Mac
   practical1.exe    # On Windows
   ```

### Example
```bash
$ cd "DS Programs/Practical 1"
$ gcc -o array_ops array_operations.c
$ ./array_ops
```
